Probabilidade Básica

Definição

Probabilidade representa incerteza sobre eventos.

P(A) = chance de A ocorrer

Regras Básicas

  • 0 ≤ P(A) ≤ 1
  • P(A) + P(¬A) = 1

Probabilidade Condicional

::contentReference[oaicite:1]{index=1}

Intuição

"Qual a chance de A acontecer dado que B já aconteceu?"

Teorema de Bayes

Fórmula Central

::contentReference[oaicite:2]{index=2}

Interpretação

  • P(A): crença inicial (prior)
  • P(B|A): evidência
  • P(A|B): crença atualizada (posterior)

Exemplo Médico

  • A: doença
  • B: teste positivo

Queremos saber: probabilidade de doença dado teste positivo.

Problema de Intuição

Testes médicos podem enganar!

Alta taxa de falsos positivos muda tudo.

Redes Bayesianas

O que são?

Grafos direcionados que representam dependências probabilísticas.

Estrutura

  • Nós = variáveis
  • Arestas = dependências
  • Tabelas = probabilidades condicionais

Exemplo

Chuva → Grama molhada → Escorregadio

Independência Condicional

Permite simplificar cálculos complexos.

Vantagem Principal

Reduz explosão combinatória de probabilidades

Overfitting

Problema

Modelo se ajusta demais aos dados de treino.

Sintomas

  • Alta precisão no treino
  • Baixa generalização

Soluções

  • Regularização
  • Mais dados
  • Simplificação do modelo

Seleção de Modelos

Objetivo

Escolher o modelo que generaliza melhor.

Trade-off

  • Complexidade do modelo
  • Erro de treinamento
  • Erro de generalização

Occam's Razor

Entre explicações equivalentes, escolha a mais simples.
