Vetores de Palavras

Como representamos o significado de uma palavra?

  • Possíveis definições:
    1. a ideia representada por uma palavra, frase, etc.
    2. a ideia que uma pessoa deseja expressar por meio de palavras, sinais, etc.

    Forma linguística mais comum de conceber o significado:

Por que Representar Palavras?

  • Computadores precisam de representações numéricas.
  • Palavras semelhantes devem possuir vetores semelhantes.
  • Essas representações são fundamentais em NLP moderno.

WordNet:
  • Banco de dados lexical de relações semânticas entre palavras

Limitações

  • Correto, mas depende do contexto
  • Palavras novas ou raras não são bem representadas
  • Subjetivo
  • Requer horas de trabalho para ser criado/atualizado
  • Precisamos de um novo banco para cada linguagem

Codificação One-Hot

  • Palavras como símbolos
  • Vetores extremamente esparsos.

Problema dos Vetores One-Hot

  • Alta dimensionalidade - um para cada palavra
  • Vetores extremamente esparsos.
  • Não capturam similaridade semântica.
  • "hotel" e "motel" ficam tão distantes quanto "hotel" e "gato".

Hipótese Distribucional

  • Palavras em contextos semelhantes tendem a ter significados semelhantes.
  • Base conceitual dos embeddings (espaço vetorial de dimensão inferior)

  • longa, curta, bufante
  • original, baseado em livros

Modelo de linguagem

O que é um modelo de modelo de linguagem?

  • Modelo que prevê a próxima palavra em uma sequência.
  • Exemplo: "Eu quero um copo de ___"
  • Aprende probabilidade de sequências de palavras.

Treinar um modelo de linguagem

  • Coletar estatísticas sobre a frequência de diferentes n-gramas e utilizá-las para prever a próxima palavra
  • uposição de Markov: 𝑥(𝑡+1) depende apenas das n-1 palavras anteriores
  • Contar palavras em algum grande corpus de texto
  • Aproximar probabilidades dado:

    • Problemas

    • Dependências longas: em geral n é no máximo 5
    • O ministério da cultura, que tem hoje como ministro o excelentíssmo fulano, é um ministério ____"
    • Mais dados e n maior?

    Rede Neural para modelos de linguagem
